Version 1

In this first verion of the chart I chose to use a simple bar chart representing the IMDB scores of several movies in my dataset. I didnt adjust scale color, or anything of that nature


Version 2

In this second version, I changed the chart to a dot plot so it was esier to read, and also changed the scale. Initially I thought the compression o the names on tbe bottom would be harder to read, but I fixed that problem in the final version


Version 3

In this final chart I added several interactive features one, I changed the size of the circles to corispond with the number of reviews of a movie. Then I used tooltip to show the name, score, and number of reviews for each data point when you hover over it. Finally, I made the chart interactive, so if you put your cursor over the chart and scroll, it will change the size of the axis so you can see the smaller circles and the name of the movie associated witht eh smaller circle.
